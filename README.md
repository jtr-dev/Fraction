# Fraction

WIP project template

## Libs / Tools

-   reagent
-   shadow-cljs
-   express
-   electron
-   docker

## Development

```
git clone refraction
cd refraction
code refraction
'Open in Container'
```

then open local bash terminal and run:

```
yarn
npx shadow-cljs watch main ui
```
which is the same as  
```
npm i -g shadow-cljs
npm start
```

from there you can open a local electron to view the app

```
cd refraction
electron .
```
